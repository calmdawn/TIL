## Javascript 분기

- if 또는 switch를 사용하여 분기를 구현 할 수 있습니다.

~~~
switch(변수이름){

  case A값:
  변수값이 A값일 때 실행문;
  break;
  
  case B값:
  변수값이 B값일 때 실행문;
  break;
}
~~~
