## 자바 웹 어플리케이션(Java Web Application)

- WAS에 설치되어 동작하는 웹 어플리케이션입니다.
- 정적 웹 프로그래밍 방식의 단점을 보완하고 동적인 서비스를 제공합니다.
- HTML, CSS, 이미지, JAVA로 작성된 클래스, 리소스 등이 포함됩니다.
